A compact theme for clear technical writing.

Compact Theme uses IBM Plex, restrained type sizes, tight section spacing, automatic light and dark palettes, and readable code blocks.

Core components

A small set of classes covers most editorial sites.

Use the layout classes as shown here, or keep only the colour variables and base typography. The theme has no framework dependency.

Readable

Comfortable type

Body copy stays near 17px. Headings remain compact and line lengths stay manageable.

Responsive

Useful at narrow widths

Sections collapse cleanly, navigation stays legible, and code scrolls without shrinking.

Code treatment

Scripts render clearly and copy cleanly.

Every pre > code block receives a language label and copy button. Add a standard language class to set the label.

<link rel="stylesheet" href="/assets/compact-theme.css">
<script src="/assets/compact-theme.js" defer></script>

Start using it

Copy the folder into any static site.

The stylesheet includes both themes and local IBM font files. The script adds theme switching and code-copy controls.
